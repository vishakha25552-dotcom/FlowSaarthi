# FlowSaarthi

FlowSaarthi is an AI-powered system that transforms complex government processes into simple, interactive flows.

# Features

- Upload government documents
- AI-powered explanation
- Automatic process detection
- Visual step-by-step flows
- Progress tracking

# Tech Stack

- Frontend: React, React Flow
- Backend: Node.js, Express
- OCR: Tesseract
- AI: Gemini API

# How it works

Upload → OCR → AI → Case Detection → Flow Generation → Interactive UI

# Why this project?

Government processes are often confusing and hard to navigate.

FlowSaarthi simplifies them into clear, visual steps that users can follow and track.
